# fiche-go
=====
Command line pastebin for sharing terminal output. Go version!

# Running Go app:

```
go run main.go
```

# Client-side usage

```
echo just testing! | nc 127.0.0.1 9999
```

```
cat file.txt | nc 127.0.0.1 9999
```

based on fiche, originally written in C:
https://github.com/solusipse/fiche
